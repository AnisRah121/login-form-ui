
# Login Form UI

A simple and responsive login form built using HTML and CSS.

## Features
- Clean UI with background image
- Hover effects
- Responsive layout
- Accessible form elements

## Technologies Used
- HTML5
- CSS3

## Purpose
This project was created to practice front-end fundamentals such as layout design, form styling, and user interface enhancement.

Open `index.html` in a browser.
